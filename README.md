## 🛠️ Architecture & Modern Stack

This application bypasses traditional structural limitations by using native compilers and next-generation pipelines:

*   **Frontend Engine:** React v18 (Vite Bundler for rapid HMR module swapping)
*   **Style Pipeline:** Tailwind CSS v4.0 (Utilizing the native `@tailwindcss/vite` compiler plugin, removing PostCSS runtime overhead entirely)
*   **Vector & Data Analytics:** Recharts charting engine for rendering high-density telemetry matrices
*   **Icon Registry:** Lucide React engineering iconography

## ✨ Features Built

1.  **Landing Terminal:** High-converting premium SaaS interface featuring metrics ledgers, glassmorphism panel arrays, and smooth structural transitions.
2.  **Telemetry Command Center (Dashboard):** Real-time monitoring of user index metrics, active engineering streaks, and chronologically mapped action timelines.
3.  **Resume Vector Parser:** Simulates dynamic keyword saturation testing and pattern alignment against modern Applicant Tracking System (ATS) structures.
4.  **Capability Gap Matrix:** Renders dynamic polar radar models mapping a developer's real-time skills profile against custom criteria baselines.
5.  **Dynamic Engineering Path (AI Roadmap):** Sequential tracking mechanics illustrating prerequisite training timelines and system project checkpoints.
6.  **AI Openings Matcher (Internships):** Evaluates live role targets and processes compatibility weight percentages for seamless deployment.
7.  **AI Interview Simulator:** Sandboxed voice and context evaluation platform imitating real-world technical vetting scenarios.
8.  **Performance Tracking & Profile Identity:** Features profile data management, localized storage status, and unified XP calculation layers.
